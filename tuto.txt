Créer le répertoire :
git clone https://github.com/aborsu985/Groupe5.git

Récupérer ce qui est en ligne et mettre à jour le répertoire local:
git fetch https://github.com/aborsu985/Groupe5.git
git pull https://github.com/aborsu985/Groupe5.git

Enregistrer vos modifications locales en local
git add fichiermodifie

Creer une contribution avec ces modifications
git commit -m "message de commit"

Synchroniser avec la version en ligne
git pull https://github.com/aborsu985/Groupe5.git
